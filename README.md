# 学者简介
**Gongxian's Homepage**

曾功贤，副教授，博士生导师。2020年博士毕业于香港大学，同年3月加入鹏城实验室任助理研究员。长期从事密码理论和应用研究，当前主要研究方向为零知识证明、通信安全与认证，相关研究成果发表在领域内顶会（ASIACRYPT、EUROCRYPT）。
